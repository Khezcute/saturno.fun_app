Qué es Saturno Bares y Restaurantes?
Un sistema de agenda de reservas
Tus clientes pueden elegir una mesa y reservarla. El sistema automáticamente administra las mesas de cada sector y los tickets agendados. Si un cliente solicita una mesa para un gran número de personas y ninguna de las mesas dentro del sector que solicita puede cumplimentar ese requerimiento, tendrá la opción de enviar un pedido de mesa especial, el administrador debe asignar las mesas y aprobar la solicitud.
Un sistema de cola virtual
Tu cliente puede solicitar una mesa desde cualquier lugar en forma online y esperar en un lugar de preferencia. Esto le permite evitar lugares cerrados con congestión de personas o esperar a ser llamado en la puerta de tu comercio. Cuando llega su turno y el sistema encuentra una mesa disponible que se ajuste a su solicitud, le asigna la mesa y lo notifica.
Un sistema de gestión de mesas
Saturno es también un sistema de administración de mesas. El administrador del bar/restaurante puede crear usuarios con perfil de camareros/as quienes a través de sus móviles pueden tomar un sector y gestionar los estados de las mesas y las solicitudes de los clientes en tiempo real.
Un sistema de calificaciones
Para cada sector creado que conforme tu comercio, podés crear items de calificaciones para que tus clientes puedan calificar de 1 a 5 estrellas lo que necesites que sea calificado. Esta encuesta se le ofrece al cliente al finalizar su sesión de mesa. Esto te permite analizar y supervisar los tiempos de atención y los feed-backs de tus clientes.
Una página Web para tu comercio
Cuando un cliente encuentra tu bar / restaurante, ingresa a la página web de tu comercio, donde podes subir la historia del lugar o cualquier texto de presentación para recibirlo, fotos del lugar y el logo de tu bar o restaurante. En esta página tu cliente tendrá disponible un formulario para solicitar una mesa.
No obtenemos datos de tus clientes
El sistema no les solicita en ningún momento sus números de teléfono. Utilizamos una tecnología que nos permite crear canales propios y privados de comunicación con tu cliente que viven sólo mientras dura la sesión del ticket y se destruyen en el momento que se cierra la sesión y el cliente se retira del comercio.
No es necesario descargar una App
Tus clientes no están obligados a ingresar a un sistema de distribución de aplicaciones como una Playstore o Appstore para descargar e instalar una App, lo que casi siempre hace que desistan de utilizarla. En el momento de ingresar a la web se les ofrece instalar una PWA que una webapp extremadamente ligera. Si tu cliente NO desea instalar la app puede omitir la instalación y continuar utilizando el sistema como una página web estándar.
Sistema de creación de tickets de contingencia
Si tu cliente no dispone de un dispositivo móvil para pedir una mesa o simplemente no desea utilizar este sistema, los camareros/as diponenen de un sistema de generación de tickets de contingencia para ingresarlos al sistema y gestionar sus mesas.
Es gratuito
No cobramos ni cobraremos membresía por utilizar este servicio. Estaremos muy atentos a recibir sugerencias que ingresen en un proceso de mejora contínua para lograr que el sistema se adapte cada día lo mejor posible a las necesidades de tu negocio y tus clientes.
